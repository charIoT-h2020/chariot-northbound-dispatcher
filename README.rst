=============================
Chariot Northbound Dispatcher
=============================

|epl|_

Northbound dispatcher of the Chariot Project

Build docker images
-------------------

.. code-block:: console
    docker build --tag chariot_northbound_dispatcher .


Features
--------

* Actuator subscription
* Message forwarding
